## Pre Requisities

- Node JS
- MongoDB
- IDE

## Run Locally

Clone the project

```bash
  git clone https://github.com/upendraTRL/sihui.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the app

```bash
  node app.js
```
